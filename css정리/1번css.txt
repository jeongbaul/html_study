# css 출처

### 브라우저 기본 스타일

---

브라우저들마다 기본적으로 지정하고있는 스타일

예: 크롬 사파리 파이어폭스 등

---

### 브라우저 사용자 스타일

---

사용자 개인의 설정으로 변경하여 더 편하게 읽을 수 있도록 한다

---

### 제작자 스타일

---

사용자가 작성한 스타일 시트

---

# css 출처 적용 우선순위

사용자  !important>제작자!important>제작자>사용자>브라우저

※주의사항 !important는 폭포의 흐름을 깰수있으니 조심

# cascading 뜻

cascading Style Sheet에서 cascading 은 폭포라는 뜻
html 문서는
제작자 스타일 우선 적용→ 그다음 브라우저 사용자 스타일 →  마지막으로 브라우저 기본스타일 적용
이처럼 cascading 의 뜻인 폭포와 같이 스타일이 우선순위에 맞게 연속적으로 적용됨을 의미한다.
